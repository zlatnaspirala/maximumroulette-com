Hang3d story's

This is for now demo game template for Mobile TPS unreal engine 4.17 version. Multiplayer shooter game with
zombies.This is free demo also new version of Hang space. In near future Hang space comes with windows, mac OS and
iOS version. You can play with others client with different platform. In LAN or internet protocol. You can open host
and wait for others or get ip address of host and connect to him.
Important : You must for now exchange ip address client - host game.
In next version i will create automate 'active' gameplay list.
Important note: zoom is disabled in this release.
Good luck!
